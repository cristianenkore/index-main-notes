# index-main-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能用于展示笔记、工具界面、交互原型或其他静态内容。仓库本身不针对任何特定的域名或网站，仅作为通用 HTML 文件存储与发布空间。

## 目录说明

```
index-main-notes/
├── index.html          # 主入口页面（如有）
├── pages/              # 存放各类独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要存放独立的 HTML 文件，每个文件可含自包含的样式与脚本。
- `assets/`：存放页面引用的公共资源，按类型分目录管理。

## 页面归档说明

- 每个 HTML 页面均为独立可运行的文件，不依赖外部服务器环境（除必要的 CDN 库外）。
- 页面内部可包含样式与脚本，推荐使用相对路径引用 `assets/` 下的资源。
- 文件名建议使用英文小写、短横线连接，便于 URL 访问。
- 若页面需要特殊运行环境（如 WebGL、Canvas 等），请在文件头部注释说明。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并确保所有资源引用路径正确。
- 更新页面：直接修改对应 HTML 文件，注意保持向后兼容。
- 删除页面：从仓库中移除文件，并检查是否有其他页面引用该文件。
- 不建议在页面中硬编码外部链接或敏感信息，以免归档后失效。
